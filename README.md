# City Garden Guys Operations Demo

This folder contains a dependency-free static demo of the City Garden Guys operations workspace, customer estimate flow and public project portal.

Run the included local server:

```bash
node server.js
```

Then visit:

- `/` for the streamlined back office manager workspace
- `/estimate` for the customer estimate request flow
- `/project/CGG-2026-00124` for a public customer project page

The manager workspace includes project queue, metrics, status control, team assignment, quote control, internal notes, uploaded areas and portal preview. The customer flow still creates an instant estimate and public project URL.
